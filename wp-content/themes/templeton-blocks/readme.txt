== Description ==

Templeton Blocks is a modern block theme built specifically for Templeton Academy. This Full Site Editing (FSE) theme leverages WordPress 6.9's latest block editor features to create a dynamic, educational website experience.

**Key Features:**

* Full Site Editing (FSE) support
* Custom color palette with 12 brand colors
* Adobe Fonts integration (Nexa, Nexa Text)
* Responsive design with mobile-first approach
* Custom block patterns for reusable content
* Optimized for educational institutions
* Dynamic year shortcode for footer copyright
* Custom navigation and button styles
* Video embed optimization
* Iframe loading animations


== Installation ==

1. Upload the theme folder to /wp-content/themes/ or install via WordPress admin
2. Activate the theme through the 'Appearance' menu in WordPress
3. Customize via the Site Editor (Appearance > Editor)
4. Configure navigation menus and add your content


== Customization ==

**Color Palette:**
The theme includes a custom color palette defined in theme.json:
- Primary Blue (#455be2)
- Primary Purple (#822f92)
- Primary Teal (#018e75)
- Dark Gray (#262626)
- And 8 additional accent colors

**Typography:**
- Heading Font: Nexa (Adobe Fonts)
- Body Font: Nexa Text (Adobe Fonts)
- Font sizes use fluid typography for optimal readability across devices

**Block Patterns:**
Access custom patterns in the block inserter:
- Feature Card: Centered card with image, label, heading, and description

**Template Parts:**
- Header: Logo, navigation menu, and CTA button
- Footer: Logo, navigation, social links, copyright

== Utility Classes ==

The theme includes utility classes for quick styling:

**Spacing:**
- .mb-1, .mb-1-5, .mb-2, .mb-3 (margin bottom)
- .mt-1-5 (margin top)
- .m-0 (no margin)
- .p-0 (no padding)
- .section-padding-standard, .section-padding-large

**Layout:**
- .no-spacing (remove all spacing)
- .no-gap (remove gaps)
- .is-reverse-mobile (reverse column order on mobile)

**Background Colors:**
- .bg-blue, .bg-purple, .bg-teal, .bg-dark, .bg-pink, .bg-white

**Button Styles:**
- .button-teal, .button-purple, .button-white, .button-yellow
